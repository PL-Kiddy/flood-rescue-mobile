
## Typography Hierarchy

### Headers
```
Screen Title:      32px, Weight 900, Color: Primary
Section Header:    20px, Weight 900, Color: Dark
Label:             14px, Weight 700, Color: Gray600
```

### Body Text
```
Input Placeholder: 16px, Weight 500, Color: Gray400
Button Text:       14px, Weight 900, Color: White/Dark
Helper Text:       12px, Weight 500, Color: Gray500
```

## Component Styles

### Button States
```
Normal:    Background color, border-radius: 12px
Pressed:   scale(0.98), opacity change
Disabled:  opacity: 0.5, color: gray
```

### Input Field States
```
Normal:    border: 1px #DDD, bg: #FAFAFA
Focused:   border: 2px primary color, shadow
Error:     border: 2px #D32F2F, bg: #FFF5F5
```

### Cards
```
Border:    1px solid #DDD
Radius:    12px
Padding:   16px
Shadow:    elevation: 2 (Android)
```

## Interactive Elements

### Buttons
```
Height:            48-56px
Padding:           Horizontal: 16px, Vertical: 12-14px
Border Radius:     12px
Font Size:         14px Bold
Touch Target:      Min 48x48px
```

### Input Fields
```
Height:            48px (single line)
Padding:           12px horizontal, 12px vertical
Border Radius:     12px
Font Size:         16px Medium
Placeholder:       Gray400
```

### Checkboxes
```
Size:              20x20px
Border Radius:     4px
Border:            1px solid
Checked BG:        Primary color
```

## Layout Grid

### Safe Area Margins
```
Top:               24px (status bar)
```

### Form Column Grid

- Title next to icon
- Right side: profile/actions
